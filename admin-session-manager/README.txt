Admin Session Manager allows WordPress admins to monitor, control, and terminate user sessions directly from the dashboard for enhanced security.

== Description ==

Admin Session Manager is a powerful WordPress plugin designed to give site administrators full control over user sessions directly from the WordPress dashboard. With this plugin, admins can monitor active user sessions, identify users currently logged into the site, and terminate sessions when necessary to ensure the security and performance of the site.
A few notes about the sections above:

== Installation ==

Search for "admin session manager" in WordPress plugins directory and Install. Or alternatively download and extract the zip content of admin-session-manager.php in your WordPress plugins directory.

e.g.

1. Upload `admin-session-manager.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where can I see the users after plugin activation? =

After plugin is installed and activated, you will see a new dashboard widget on your WordPress dashboard.
